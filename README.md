# fonbet-ipbl

Лёгкий сигнальный бот по баскетболу IPBL (Fonbet). Сигнал ТМ на весь матч в большом перерыве.
Управление — кнопочный Telegram-бот. Без веб-панели.

## Как работает

В перерыве (после Q2: Pro — таймер 20:00, Prime — 24:00) фиксируем сумму очков и линию ТМ матча
(самый высокий кф в 1.95–2.1). Если `2×сумма − линия ≤ −16` — шлём сигнал ТМ с этой линией.
По дивизиону Prime дополнительно — безусловное инфо-уведомление в перерыве.

Стратегии (свой chat_id, окно работы по МСК и статистика у каждой):
- **signal_tm** — сигнал по формуле (все 4 лиги, 1 чат)
- **prime_info** — инфо в перерыве по Prime

Статистика: баланс старт 100000₽, ставка 1000₽/сигнал, прибыль = +1000×(кф−1) / −1000.
Сигналы без линии (void) в прибыль не идут. БД — только сигналы, 1 строка на матч.

## Запуск

```bash
cd ~/projects/fonbet-ipbl
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# токен можно задать через env (по умолчанию зашит в config.py)
export BOT_TOKEN="..."

python bot.py        # бот управления (Запуск/Стоп парсера, чаты, время, статистика)
# парсер запускается кнопкой из бота; вручную для дебага:
python parser.py            # живой мониторинг в консоли
python parser.py --once     # один цикл и выход
python parser.py --reset    # очистить БД и запустить
```

chat_id и окна работы задаются кнопками в боте (`/start`).
Детали API — в `../../claude-project/fonbet-api-research.md` (раздел IPBL).
